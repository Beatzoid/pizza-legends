# Pizza Legends

A Pokemon-style game centered around a pizza chef

# How to play

WASD/Arrow keys to move, enter to interact

# Running Locally

1. `tsc -w` to start the Typescript watcher
2. `npm i -g serve && serve` to start the local server
3. Visit `http://localhost:3000`, replacing 3000 with the port of the server if its different
