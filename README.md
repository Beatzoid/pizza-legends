# Pizza Legends

A pokemon-style pizza-themed RPG written in Typescript

# Demo

[Here](https://pizza-legends.up.railway.app/)

# Usage

```bash
git clone https://github.com/beatzoid/pizza-legends
cd pizza-legends
yarn install
yarn watch
# In a new terminal
yarn dev
```

And open the URL in your browser
